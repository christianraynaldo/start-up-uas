# ReWear Studio Website

Website statis untuk konsep bisnis ReWear Studio berdasarkan Business Model Canvas.

## Cara Menjalankan

1. Extract file ZIP.
2. Buka folder `rewear-studio-web`.
3. Klik dua kali file `index.html`.
4. Website langsung terbuka di browser.

## File

- `index.html` = struktur halaman website
- `style.css` = desain dan responsive layout
- `script.js` = menu mobile, animasi scroll, dan form alert

## Catatan

Website ini sudah responsif untuk desktop dan mobile.
